# NestJS + GraphQL + TypeORM

## Description

Example backend API built with NestJS + GraphQL + TypeORM

## Installation

```bash
$ npm install
```

## Docker Compose

```bash
$ docker compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
